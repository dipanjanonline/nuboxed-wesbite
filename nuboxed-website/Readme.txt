NuBoxed Website - Ready to Deploy

To deploy this site:
1. Upload this folder to Vercel, Netlify, or GitHub Pages
2. Vercel is recommended: https://vercel.com 
3. Drag and drop the folder to deploy instantly
4. No build tools or coding required

All dependencies are loaded via CDN:
- React
- ReactDOM
- Babel (for JSX support)
- Tailwind CSS

To customize:
- Edit App.jsx with your browser-based code editor
- Re-upload the folder to your host
